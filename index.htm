Advance Login System

Version 1.5

HOW TO INSTALL:

    1. First fill in the correct values in the config.php file (the server, the username/password combination for the MySQL server And the database name.)
    2. Then run Install.php
    3. Then signup (signup.php) an useraccount, or
    4. Log-in (login.php) using the admin account (username= admin password= admin)
    5. Remove install.php
    6. Change the admin password
    7. vote & give feedback on http://www.planetsourcecode.com/vb/scripts/ShowCode.asp?txtCodeId=871&lngWId=8

HOW TO UPDATE:

Update from version 1.4 or below:

overwrite all the (changed files) which are used by Advance Login System
run install.php?action=update

update from version 1.2 or above:

overwrite al the (changed) files which are used by Advance Login System.

update from version 1.1:

overwrite al the (changed) files which are used by Advance Login System.
run install.php?action=update&preUpdate=true

update from an version before 1.1 (if there is no version number anywhere there is a big chance it is before 1.1):

overwrite al the (changed) files which are used by Advance Login System.
there isn't really a way to update. The best way is to remove the table (query: DROP TABLE signup;)
and run install.php
